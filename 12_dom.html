JS DOM manipulation

what are the capabilities of javascript on the browser.

  1. Reading , removing or adding an elelemnt's text content
  2. Creating new HTML elements and dynamically adding them to the page.
  3. Style an element using the style attribute
  4. Manipulate other attributes of an element
  5. Manipulate list of classes (classlist) of an element
  6. Responding to user interactions like clicks, scrolls, or form submission(event listeners)
  7. Making requests to servers
  8. Processing data recieved from the server
  9. Interacting with browser functionalities like geolocation or local storage.
  10. Input validation
